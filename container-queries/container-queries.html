Container Queries

Container queries are a new CSS feature that allow you to change the styling of an element based on the size of its container. This is useful for creating responsive layouts that are based on the size of the viewport, rather than the size of the device.

Container queries are currently only supported in Firefox, but there is a polyfill that can be used to add support to other browsers.
